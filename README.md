# Wecle's Skills

A curated collection of [Agent Skills](https://agentskills.io/home) reflecting [Wecle](https://github.com/Wecle)'s preferences, experience, and best practices.

## Installation

```bash
pnpx skills add Wecle/skills --skill='*'
```

Or install globally:

```bash
pnpx skills add Wecle/skills --skill='*' -g
```

Or install a specific skill:

```bash
pnpx skills add Wecle/skills --skill=delegating-to-codex
```

Learn more about the CLI usage at [skills](https://github.com/vercel-labs/skills).

## Skills

| Skill | Description |
|-------|-------------|
| [delegating-to-codex](skills/delegating-to-codex) | Delegate coding, debugging, and implementation tasks to Codex CLI. The agent acts as tech lead: researches requirements, crafts prompts, supervises execution, and reviews output. |
